# Cypress

## 기능
* 테스트 설정
* 테스트 작성 
* 테스트 실행 
* 디버그 테스트 

## 지원
* 통합 테스트
* 단위 테스트
* 종단 간 테스트

설치
```
  npm i
```

실행
```
  npm run cypress:open
  npm run lint
  npm run lintfix
```

지원되는 브라우저
```
  https://docs.cypress.io/guides/guides/launching-browsers.html#Browsers
```
API 공식 Document
```
  https://docs.cypress.io/api/
```
